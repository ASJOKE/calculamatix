Comprendre les composants PC en 2026

Pourquoi les composants doivent-ils être compatibles entre eux ?
Un PC fonctionne en chaîne — chaque composant communique avec les autres via la carte mère. Trois compatibilités critiques : (1) Socket CPU/CM : un Ryzen 9000 ne fonctionne QUE sur socket AM5 (B650, B850, X670E, X870E). Un Intel Core 14000 sur LGA1700, un Core Ultra 200 sur LGA1851. (2) RAM : DDR4 ou DDR5 selon la carte mère, jamais les deux. (3) Alimentation : doit fournir assez de watts pour CPU + GPU + 30% de marge. Notre configurateur détecte automatiquement ces 3 incompatibilités et vous alerte en rouge.
Qu'est-ce qu'un goulot d'étranglement (bottleneck) ?
C'est quand un composant limite les performances d'un autre. Exemple : RTX 5090 à 2500€ avec un Ryzen 5 5600 à 130€ — le CPU est trop faible, il n'arrive pas à "alimenter" le GPU en données. Résultat : tu paies 2500€ une carte qui tourne à 60% de ses capacités. À l'inverse : Ryzen 9 9950X3D avec une RTX 5060 — le CPU s'ennuie, le GPU est saturé. Règle d'or 2026 : pour le gaming, le prix du GPU doit être 1,5 à 2,5× celui du CPU. Notre configurateur détecte ces déséquilibres et te prévient.
À quoi sert chaque composant d'un PC ?
CPU (processeur) : le cerveau, calcule tout. Le X3D AMD est imbattable en gaming. GPU (carte graphique) : affiche les pixels, le composant n°1 pour le gaming. RAM : mémoire vive temporaire — 32 Go DDR5 6000 MHz CL30 = sweet spot 2026. SSD NVMe : stockage rapide, charge OS et jeux. Carte mère : connecte tout. Alimentation (PSU) : fournit l'électricité — JAMAIS d'occasion. Boîtier : héberge tout, gère le refroidissement par airflow. Refroidissement : ventirad ou AIO (watercooling tout-en-un) pour le CPU.

Quelle alimentation choisir et combien de watts ?
Calcul simple : TGP du GPU + TDP du CPU + 100W pour les autres composants, puis ajoutez 30% de marge. Exemples : RTX 5060 + Ryzen 5 7600 = 145+65+100 = 310W → alim 500W suffit. RTX 5080 + Ryzen 7 9800X3D = 360+120+100 = 580W → alim 850W. RTX 5090 + Ryzen 9 9950X3D = 575+170+100 = 845W → alim 1000-1100W. Privilégiez les certifications 80 Plus Gold/Platinum, normes ATX 3.1 avec connecteur 12V-2x6 natif (évite les adaptateurs qui ont brûlé sur certaines RTX 4090). Marques fiables : be quiet!, Corsair, Seasonic, Thermaltake.
DDR4 ou DDR5 en 2026 — vu la pénurie ?
Cas n°1 : nouveau PC → DDR5 obligatoire (les CPU Ryzen 9000 et Intel Core Ultra 200 ne supportent QUE la DDR5). Sweet spot : 32 Go 6000 MHz CL30 EXPO/XMP, ~310€. Cas n°2 : vous avez déjà DDR4 → gardez-la ! Un kit DDR4 32 Go reste à 150€ vs 310€ DDR5. Vous pouvez choisir une plateforme legacy (AM4 Ryzen 5000 ou Intel LGA1700 DDR4). Cas n°3 : budget serré, neuf → 16 Go DDR5 5600 (~160€) maintenant, upgrade plus tard. Évitez les fréquences > 6400 MHz qui forcent les CPU AMD en mode dégradé.
Quels composants acheter d'occasion sans risque ?
OUI : Boîtier (aucun risque, économies réelles), GPU si vendeur fiable et garantie restante (RTX 4070/4080 à -30%), Ventirad air, Carte mère (vérifier socket et BIOS). NON : CPU (peu d'écart neuf/occasion, garantie perdue), RAM (durée de vie limitée, prix très volatils), SSD (usure flash NAND), Alimentation (JAMAIS — risque incendie ou destruction des composants). En 2026, le meilleur deal occasion c'est le GPU : une RTX 4080 d'occasion vérifiée à 680€ au lieu de 1200€ neuve.
